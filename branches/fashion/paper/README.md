**Paper entry for the fashion branch inside the AKM mother hub.**

## Paper Position

This branch is written as a method and system-design paper for wardrobe planning and outfit decisions.
It is not presented as image-recognition validation, large-sample benchmarking, or a general aesthetics paper.

## Evidence Scope

The evidence model is `single-user long-term self-use`.
The relevant evidence comes from stable scene logic, reusable wardrobe modeling, explicit constraints, and external behavior checks.

## Current Manuscript

- [manuscript.md](./manuscript.md): branch-level markdown manuscript for the fashion line inside AKM.

## Reading Note

Read this paper as a constrained wardrobe-decision system case.
The core claim is methodological: AKM can turn styling from generic taste language into scene-aware and asset-aware planning.

## Relation to the Skill

The paper explains the method structure, evidence form, and boundaries.
The skill provides the operational prompting and execution interface.
