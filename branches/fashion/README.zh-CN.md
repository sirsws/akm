# AKM Fashion

**AKM Fashion = 先建立形象画像，再做穿搭与采购裁决。**

[English](./README.md)

这不是一个泛泛给单品建议的穿搭助手。
它解决的是更真实的问题：当体型、场景、衣橱资产和功能约束都重要时，今天到底该怎么穿，接下来到底该补什么。

## 方法结构

1. `Elicitation`
   先问清场景、体型、衣橱、风格偏好和功能约束。
2. `Record`
   把画像、核心单品、缺口和禁区沉淀成上游资产。
3. `Execution`
   读取画像后输出穿搭裁决、短板分析与采购优先级。

## 证据形态

这个分支最适合被诚实地写成：

**单用户长期自用的 AKM 场景系统设计与方法案例。**
