# Fashion Input Template

## 1. 场景

- 你最常出现的 3-5 个场景是什么？
- 哪个场景目前最需要解决？

## 2. 体型与体态

- 你的体型特征是什么？
- 有没有特别在意或想修饰的地方？

## 3. 风格偏好

- 你想传递的感觉是什么？
- 你明确讨厌哪些风格或元素？

## 4. 现有衣橱

- 你目前已有的核心上装、下装、鞋履、外套有哪些？
- 哪些是高频主力？哪些基本废置？
