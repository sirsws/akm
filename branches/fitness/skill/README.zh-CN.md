# Fitness Skill

<p align="center">

## 用途

Fitness skill 是 AKM 在训练裁决工作流中的可执行入口。
它不是单一 prompt，而是三段式方法包。

## 方法层

1. 前置挖掘
2. 结构化记录
3. 执行裁决

## 文件

- [ELICITATION_PROMPT.md](./ELICITATION_PROMPT.md)
- [RECORD_TEMPLATE.md](./RECORD_TEMPLATE.md)
- [EXECUTION_PROMPT.md](./EXECUTION_PROMPT.md)
- [SKILL.md](./SKILL.md)
- [INPUT_TEMPLATE.md](./INPUT_TEMPLATE.md)

## 输出方向

它的核心输出不是泛泛训练话术，而是基于身体限制、器械环境、时间预算、恢复状态和目标层级的训练裁决。
