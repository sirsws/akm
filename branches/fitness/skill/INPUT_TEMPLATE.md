# Fitness Input Template

## 1. 目标

- 你当前最优先的目标是什么？
- 你明确不优先的目标是什么？

## 2. 身体情况

- 当前是否有疼痛、旧伤、关节限制、脊柱限制或医生明确禁止项？
- 哪些动作你已经知道不适合自己？

## 3. 器械环境

- 你在哪训练？
- 当前可用器械有哪些？
- 有没有明确缺失的关键器械？

## 4. 时间预算

- 每周练几次？
- 单次上限多少分钟？
