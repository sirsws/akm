**AKM 母港中健身分支论文的规范入口。**

## 论文定位

这个分支按“真实约束下的健身规划方法与系统论文”来写。
它不是广泛样本的效果验证论文，也不声称对所有用户普遍有效。

## 证据口径

证据模型是 `n=1 longitudinal self-use`。
相关证据来自稳定约束下的长期使用、结构化状态记录、训练日志和外部行为测试。

## 当前稿件

- [manuscript.md](./manuscript.md)：AKM 仓中健身分支的 markdown 论文稿。

## 阅读说明

应把这篇论文读成一个受约束决策系统案例。
它的核心主张是方法论上的：AKM 能把训练建议变成 profile-first、状态依赖的规划过程。

## 与 Skill 的关系

论文负责解释方法逻辑、证据形式和边界。
Skill 负责提供可执行的提示词和操作接口。
