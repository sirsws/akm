## Original Public Paper Line

Two public paper variants define the original DaE research line:

- **arXiv-style draft:** *Dialogue as Elicitation: Building High-Fidelity Personas for Personalized AI Advisors with LLMs*
- **SSRN version:** *Reducing Alignment Debt in AI Advisory: The Dialogue-as-Elicitation Approach*

Original public assets remain in the legacy research repository:

- **Legacy research repository:** https://github.com/sirsws/DaE-Personal-Strategic-Asset
- **SSRN PDF:** https://raw.githubusercontent.com/sirsws/DaE-Personal-Strategic-Asset/main/papers/dae_ssrn.pdf
- **arXiv draft PDF:** https://raw.githubusercontent.com/sirsws/DaE-Personal-Strategic-Asset/main/papers/dae_arxiv_draft.pdf

## Current Files in This Branch

## How to Read It

1. Read the original SSRN or arXiv PDF for the standalone DaE paper line.
2. Read `manuscript.md` here for the AKM-side markdown version.
3. Return to the DaE branch entry when you want the implementation view rather than the paper view.

## Relationship to AKM

AKM is now the mother-hub framework.
DaE remains the first complete reference implementation in the persona-aware advisory scene.
The paper line therefore remains historically DaE-first, while the repository structure is now AKM-first.
